NOTES

This section is for notes on several stuffs.

User Registration

In User Registration Form, there will be "role" input. The input must consist of 1 OR 2, in which 1 = QA and 2 = Developer.

User Login

In User Login, there will be "remember" input. It's like the "remember me" option. The value must be BOOLEAN (true or false)

Create Issue

In Create Issue, there will be "severity" input. The value has to be numeric from 1 to 3 with each number describes as: 1 = Low, 2 = Medium, 3 = High

Update Issue

In Create Issue, there will be "status" input. The value has to be 0 OR 1. Each number describes as: 0 = Closed, 1 = Opened
